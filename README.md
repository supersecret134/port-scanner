# 🔐 Python Port Scanner

A fast multi-threaded port scanner built using Python. This tool scans open ports on a target system and helps identify exposed services.

## 🚀 Features
- Scans ports 1–1024
- Multi-threaded for faster performance
- Supports domain and IP input
- Clean and simple output

## 🛠️ Technologies Used
- Python
- Socket Programming
- Threading

## ▶️ How to Run

```bash
python main.py
